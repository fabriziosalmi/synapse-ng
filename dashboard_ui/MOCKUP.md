## Color Legend

- 🟢 **Green (Healthy)**: Metric at or better than target
- 🟡 **Yellow (Warning)**: Metric 30-50% off target
- 🔴 **Red (Critical)**: Metric >50% off target
- 🔵 **Cyan**: Governance/proposals
- 🟣 **Purple**: Evolution/code generation
- 🟠 **Orange**: Treasury/warnings

---

## Key Interactions

1. **Click vital sign card** → Shows historical chart (future feature)
2. **Click diagnosis card** → Expands with full details
3. **Click proposal** → Opens detailed view with vote simulator
4. **Click "Simulate Vote"** → Calculate your contextual vote weight
5. **Click timeline event** → Opens code viewer for code_upgrade proposals
6. **Click "View Code"** → Full Rust source with syntax highlighting
